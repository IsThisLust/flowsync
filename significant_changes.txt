FlowSync Significant Changes
============================

[2026-04-24] Reminder feature for task deadlines

Files updated:
- js/store.js
- dashboard.html
- tasks.html

What changed:
- Added shared reminder helpers in store.js to detect incomplete tasks that are overdue or due within 30 minutes.
- Added reminder message formatting for three cases: overdue, due very soon, and due in X minutes.
- Added a shared reminder controller that:
  - runs once on page load
  - checks every 30 seconds
  - prevents duplicate reminders in the current page session
  - queues multiple reminders 2 seconds apart
  - re-checks task state before showing a queued reminder
- Wired the reminder controller into dashboard.html.
- Wired the reminder controller into tasks.html.
- Added desktop notification support on dashboard/tasks when browser notification permission is already granted.
- Updated toast handling on dashboard/tasks so repeated reminders do not stack awkwardly.
- Added "Due soon" / "Overdue" visual badges in the dashboard AI-suggested task list.
- Added animated due-soon / overdue deadline chips in the tasks list.

Behavior notes:
- Reminders do not fire for completed tasks.
- If a task is completed or deleted after being queued, it is skipped.
- Reloading the page allows reminders to appear again if the task is still due soon or overdue.
- Interval cleanup is registered on page unload.
